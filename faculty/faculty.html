Cohort Controls

Select a Cohort

Faculty will eventually be able to create, rename, archive, and promote cohorts year to year.

Create a cohort first, then generate a resident access code for that cohort.

Manage Cohorts

Rename cohorts, archive cohorts you no longer want in active workflows, or restore archived cohorts later.

Cohorts will load after selecting an organization.

Archived cohorts are hidden from active selectors but remain available for history and restoration.

Cohort Promotion

Promote Residents

Move all active residents from one cohort into another. Attempts stay historically connected to the cohort where they were completed.

Create or select a target cohort before promoting residents.
