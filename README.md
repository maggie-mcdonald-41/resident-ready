# Resident Ready

**Resident Ready** is a resident-facing medical education dashboard designed to support board-readiness, diagnostic practice, clinical reasoning review, targeted remediation, and growth tracking.

Public product name:

**Resident Ready by McDonald Medical Education Solutions**

Internal folder/project names may still reference `doctor-dashboard` during development.

---

## Project Purpose

Resident Ready helps medical residents move through a complete learning cycle:

1. Complete a diagnostic assessment
2. Review board-readiness results
3. Identify strengths and priority review areas
4. Practice targeted weak areas
5. Review clinical reasoning explanations
6. Track diagnostic and practice growth over time

The current focus is the **resident view first**. Faculty and cohort analytics are planned for a later phase.
